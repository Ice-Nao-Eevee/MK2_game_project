# ❄️ MK2 Game Project - Platformer Adventure

[![Unity Version](https://img.shields.io/badge/Unity-2022.3+-blue.svg?style=flat&logo=unity)](https://unity.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![School](https://img.shields.io/badge/School-SMK_Telkom_Purwokerto-red.svg)](https://smktelkom-pwt.sch.id/)

Sebuah game platformer 2D seru yang dikembangkan sebagai bagian dari proyek pembelajaran di jurusan **Software and Game Development (PPLG)**. Jelajahi level, kumpulkan buah, dan hindari rintangan mematikan!

---

## 🚀 Fitur Utama
* **Smooth Platforming:** Mekanik pergerakan karakter yang responsif.
* **Challenging Obstacles:** Berbagai jebakan seperti *DeathBox* dan duri yang menantang.
* **UI Systems:** Menu kemenangan (*Finish*) dan menu kegagalan (*Failed*) yang terintegrasi.
* **Level Progression:** Sistem perpindahan antar level yang dinamis.

## 🛠️ Teknologi yang Digunakan
* **Engine:** Unity Engine
* **Bahasa Pemrograman:** C# (C-Sharp)
* **Design & UI:** Figma & Unity UI System
* **Version Control:** Git & GitHub

## 🎮 Cara Bermain
1. Unduh hasil build terbaru di bagian [Releases](https://github.com/Ice-Nao-Eevee/MK2_game_project/releases).
2. Ekstrak file `output.zip`.
3. Jalankan `Home Screen.exe`.
4. Gunakan **Arrow Keys** atau **WASD** untuk bergerak dan melompat.

## 📁 Struktur Folder Penting
* `/Assets`: Berisi seluruh kodingan (Scripts), Sprites, dan Prefabs.
* `/Assets/Scripts`: Tempat logika utama game (termasuk `FinishManager.cs` dan `DeathManager.cs`).
* `/ProjectSettings`: Konfigurasi engine Unity.
